A growing list of Beaver Builder modules. The free version contains the following modules:

* Alerts
* Animated Button
* Breadcrumbs
* Button
* Card
* Copyright
* Gists
* Gravatars
* Instagram feed
* Photo Overlay
* Social Media Icons
* Vegas Full-width Slideshow
* Syntax Highlighter
* Separators and Spacers

> <a href="https://bbvapormodules.com">Upgrade to Pro for more Beaver Builder Modules</a>

== Installation ==

1. Go to Plugins->Add New in your WordPress Dashboard and search for 'Vapor Modules'. If you can't find it, you can download the plugin from here and upload it by going to Plugins->Add New->Upload plugin.

2. Make sure Beaver Builder is installed

== FAQ ==
